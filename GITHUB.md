# MH Studio GitHub Repository

**Repository:** https://github.com/millerhughes/MHStudio

This web-dev-site folder is synced with the standalone MHStudio repo on GitHub. All changes committed here should be pushed to that repo.

## Push workflow

```bash
git push origin master
```

The repo is set to auto-deploy on Vercel when changes are pushed to the main branch.

## Quick reference

- **GitHub URL:** https://github.com/millerhughes/MHStudio
- **Live site:** https://mhughesstudio.co.uk
- **Branch:** master (main branch)
